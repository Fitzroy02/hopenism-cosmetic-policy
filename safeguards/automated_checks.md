# AUTOMATED CHECKS LOGIC

The marketplace uses automated systems to prevent unsafe or non-compliant products from being listed.

## 1. Ingredient Scanner

- banned ingredient detection
- restricted ingredient thresholds
- allergen detection
- nanomaterial detection
- pH range validation

## 2. Claim Scanner

Flags:

- medical claims
- borderline claims
- misleading claims
- unsubstantiated claims

## 3. Label Compliance Scanner

Checks for:

- INCI format
- batch number
- PAO symbol
- warnings
- Responsible Person
- net content
- country of origin

## 4. Documentation Validator

Ensures:

- CPSR present
- PIF present
- Responsible Person verified
- safety testing complete

## 5. Risk-Based Routing

High-risk categories → human review
Low-risk categories → automated approval
